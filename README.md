# To-Do-List

📝 To-Do List App

A simple interactive To-Do List built with HTML, CSS, and JavaScript.
Add, complete, and delete tasks dynamically with a clean, responsive UI.

🔹 Features

 - Add Tasks: Type a task and add it to the list.
 - Complete Tasks: Mark tasks as done with a check button.
 - Delete Tasks: Remove tasks from the list easily.
 - Responsive Layout: Centered container with clean styling.
 - Interactive Buttons: Hover effects and visual feedback for buttons.

---

💻 Technologies Used

 - HTML5 – structure and layout
 - CSS3 – styling and design
 - JavaScript (Vanilla) – functionality and interactivity
 - Font Awesome – icons for buttons

---

🚀 How to Use

 1. Clone or download the repository.
 2. Open index.html in your browser.
 3. Type a task in the input box and click the Add (+) button.
 4. Mark tasks as complete with the check button.
 5. Remove tasks with the trash button.
